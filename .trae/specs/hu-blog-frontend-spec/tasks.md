# hu-blog-frontend - 实现计划（分解和优先级任务列表）

## [ ] 任务 1：项目设置和配置
- **优先级**：P0
- **依赖**：无
- **描述**：
  - 初始化 umi + ant-design-pro 项目
  - 配置 API 基础 URL
  - 设置 TypeScript 配置
- **涉及的验收标准**：N/A（设置任务）
- **测试要求**：
  - `programmatic` TR-1.1：项目初始化无错误
  - `programmatic` TR-1.2：API 基础 URL 配置正确
- **注意**：使用 umi 的官方脚手架工具创建项目

## [ ] 任务 2：认证基础设施
- **优先级**：P0
- **依赖**：任务 1
- **描述**：
  - 实现 JWT 令牌存储/检索
  - 创建认证上下文/提供者
  - 设置请求拦截器以添加认证头
- **涉及的验收标准**：AC-2, AC-3
- **测试要求**：
  - `programmatic` TR-2.1：JWT 令牌存储在 localStorage 中
  - `programmatic` TR-2.2：认证头被添加到认证请求中
  - `programmatic` TR-2.3：登出时令牌被清除
- **注意**：使用 umi 的请求库进行 API 调用
